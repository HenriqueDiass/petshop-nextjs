# 🐾 PetDev - Landing Page de Pet Shop

Uma landing page moderna, performática e totalmente responsiva desenvolvida para o setor de cuidados animais. O projeto utiliza as tecnologias mais recentes do ecossistema React para entregar uma interface fluida e profissional.

## 🚀 Tecnologias

Este projeto foi desenvolvido utilizando:

- **Next.js** (App Router)
- **TypeScript** para tipagem estática e segurança de código
- **Tailwind CSS** para estilização utilitária e responsiva
- **Componentes Modulares** para fácil manutenção

## 📂 Estrutura do Projeto

Baseado na arquitetura atual:

- `src/app/`: Contém as rotas e o layout principal da aplicação.
- `src/app/_components/`: Componentes específicos da página inicial (`Hero`, `About`, `Services`, `Testimonials`, `Footer`).
- `public/`: Assets estáticos, incluindo fotos e logos de parceiros (Royal Canin, Whiskas e Premier).

## 🛠️ Instalação e Execução

Para rodar o projeto localmente, siga os passos:

1. **Clone o repositório:**

       git clone https://github.com/HenriqueDiass/petshop-nextjs.git

2. **Acesse a pasta do projeto:**

       cd petdev

3. **Instale as dependências:**

       npm install

4. **Inicie o servidor de desenvolvimento:**

       npm run dev

5. **Acesse no navegador:**
   Abra [http://localhost:3000](http://localhost:3000)
